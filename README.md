# Simple API

Api created to manage users in the application

## Instructions
- Copy the .env.example file to .env
- run  ```docker-composer up -d``` in the docker folder

## Routes
- Index: ```GET /users```
- Show: ```GET /users/{id}```
- Store: ```POST /users```
- Update: ```POST /users/{id}```
- Delete: ```GET /users{id}```
